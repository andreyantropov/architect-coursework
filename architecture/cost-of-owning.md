# Расчёт стоимости владения системой 

## Базовые предположения

| Параметр | Значение |
|---------|----------|
| Начальное количество пользователей | 500K (пилот + early adopters) |
| Год 1 | 1.5M активных пользователей |
| Год 2 | 2.5M |
| Год 5 | 3M+ |
| Среднее число тренировок на пользователя в месяц | 8 |

| Использование AI/ML модели | ~100K инференсов в день |
| Облако | Multi-cloud: AWS + GCP |
| Количество микросервисов | ~10–12 core сервисов |
| Поддерживаемые регионы | EU, US, APAC |

# Год 1: Запуск и масштабирование MVP

## Фокус:
- Завершение MVP и первого релиза
- Настройка CI/CD и мониторинга
- Первичное распределение по регионам
- Тестирование нагрузки и failover

### Примерная стоимость: **$1.2 – $1.5 млн**

| Статья расходов | Сумма ($) |
|------------------|-----------|
| Cloud (AWS/GCP): compute, DB, storage | ~450K |
| CDN, API Gateway, Event Bus | ~100K |
| DevOps и SRE команда (5 человек) | ~300K |
| Разработка и QA (команда из 12) | ~900K |
| Безопасность и compliance | ~70K |
| AI/ML (инференс, обучение) | ~80K |
| Лицензии и инструменты | ~50K |
| Disaster Recovery (резервный DC) | ~50K |
| **ИТОГО** | **~$2.1 млн** |

> В первый год затраты выше из-за создания MVP, найма команды и настройки всей системы.

## Фокус:

- Расширение AI/ML функциональности

| Статья расходов | Сумма ($) |
|------------------|-----------|

| CDN, API Gateway, Event Bus | ~150K |
| DevOps и SRE команда (6 человек) | ~360K |
| Разработка и QA (команда из 15) | ~1.1M |
| Безопасность и compliance | ~100K |
| AI/ML (модели, инференс) | ~150K |
| Лицензии и инструменты | ~80K |
| Disaster Recovery (поддержка) | ~50K |
| **ИТОГО** | **~$2.79 млн** |

> Удельная стоимость на пользователя снижается благодаря оптимизации и автоматизации.

# Год 5: Зрелая система, фокус на удержании и монетизации

## Фокус:
- Достижение 3M+ пользователей
- Полноценная монетизация (магазин, подписки)
- Перевод части сервисов в Serverless
- Обновление AI-моделей и персонализации
- Улучшение производительности и отказоустойчивости

### Примерная стоимость: **$2.5 – $3 млн**

| Статья расходов | Сумма ($) |
|------------------|-----------|
| Cloud (compute, БД, storage) | ~1.1M |
| CDN, API Gateway, Event Bus | ~200K |
| DevOps и SRE команда (8 человек) | ~480K |
| Разработка и QA (команда из 18) | ~1.3M |
| Безопасность и compliance | ~120K |
| AI/ML (обучение, инференс, A/B тесты) | ~250K |
| Лицензии и инструменты | ~100K |
| Disaster Recovery (full multi-cloud) | ~100K |
| **ИТОГО** | **~$3.65 млн** |

> Ежегодный прирост TCO — около **15–20%**, но рост выручки от монетизации компенсирует это.

## Анализ эффективности и возможные экономии

| Область | Возможные улучшения |
|--------|----------------------|
| **Cloud Optimization** | Использование Spot Instances для batch-задач, переход части workload в serverless |
| **AI/ML** | Кэширование популярных инференсов, использование ONNX для унификации моделей |
| **Безопасность** | Автоматизация аудита, внедрение Zero Trust Architecture |
| **Команда** | Увеличение доли remote сотрудников, усиление внутреннего обучения |
| **Инфраструктура** | Использование open source решений вместо vendor lock-in (например, self-hosted ClickHouse вместо BigQuery) |
